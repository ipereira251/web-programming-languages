Comet Commerce

Comet Commerce is a simple e-commerce website that adheres to the OWASP Top 10, and has the beginnings of a Runtime Application Self-Protection or RASP system. Rather than other e-commerce sites where vendors list their products, comet commerce flips this on its head and instead intends to serve as a secure platform for customers to commission products from vendors. Learn More

  • HTTPS support.
  • CORS policy established.
  • Event logs for added security.

PHI Redactor: De-identification Tool

A Python-based tool designed to automatically remove and restore Protected Health Information (PHI) from medical records using regular expressions. This app is built with Streamlit for a simple interface, allowing users to upload medical text files, de-identify them by redacting sensitive information, and later re-identify them by restoring original content using saved mappings. Learn More

  • Simple browser interface.
  • Removes all 18 identifiers under HIPAA.
  • Support for de- and re-identification.
